# Visual Prompts

## Stitch / Google

Use este prompt para recriar a tela em Stitch:

```text
Create a luxury editorial beauty booking landing page for an eyebrow design professional named Ellen Martins Brows. Use a warm nude cream background, espresso brown typography, terracotta accents, asymmetrical hero layout, serif display headlines, floating glass appointment cards, service cards for eyebrow design, brow lamination, natural henna and brow revitalization, plus a secure booking form. The tone should feel premium, human, Instagram-ready and mobile-first. Avoid generic purple SaaS visuals.
```

## ChatGPT Images

Use este prompt para gerar imagens originais, sem copiar perfis reais:

```text
Create an original editorial beauty image for an eyebrow design studio landing page. Close-up artistic portrait focused on elegant natural eyebrows, warm cream and terracotta palette, soft studio light, premium Brazilian beauty brand mood, clean skin texture, no text, no logo, no resemblance to any real person, suitable for a public website hero section.
```

## Observacao

As imagens e textos do Instagram devem servir apenas como referencia de mercado. Nao reutilizar fotos, marcas, legendas ou identidade visual de terceiros sem permissao.
